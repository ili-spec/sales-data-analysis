# Sales Data Analysis Project

## Overview
This project analyzes retail sales data to uncover key business insights,
customer behavior patterns, and sales trends over time.

The goal is to demonstrate a complete data analysis workflow:
data loading, cleaning, exploratory data analysis, and KPI reporting.

## Dataset
- Source: Online Retail transactional dataset
- Format: Excel / CSV
- Records include invoices, products, customers, quantities, prices, and countries.

## Project Structure
- data/raw/            : Original dataset
- data/processed/      : Cleaned and processed data
- data/reports/        : Business insights and reports
- notebooks/           : Jupyter notebooks for analysis
- README.md            : Project documentation

## Analysis Steps
1. Data loading
2. Data cleaning
3. Exploratory data analysis (EDA)
4. KPI and business insights
